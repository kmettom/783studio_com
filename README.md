# Tomas Kmet - portfolio - Nuxt 3 | ThreeJS | GSAP

## Setup

Install the dependencies:

NODE VERSION: 22

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## Development Server

Start the development server on http://localhost:3000

```bash
npm run dev
```

## Production

Build the application for production:

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```
